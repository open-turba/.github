Turba Open Lab builds open-source datasets, models, and tools for mineral, biological, and digital plant fertilization in Morocco.

## Repositories

| Repository | Purpose |
|---|---|
| [`turba-client`](https://github.com/open-turba/turba-client) | Python client and CLI for querying site profiles, crops, and NPK recommendations. |
| [`turba-data`](https://github.com/open-turba/turba-data) | Dataset package for loading analysis-ready fertilizer recommendation datasets. |
| [`turba-models`](https://github.com/open-turba/turba-models) | Model package for loading pretrained fertilizer recommendation models. |
| [`turba-simulation`](https://github.com/open-turba/turba-simulation) | Crop simulation package for the Turba ecosystem [Coming Soon]. |
| [`skills`](https://github.com/open-turba/skills) | Agent-facing instructions and workflows for using Turba tools consistently. |
